跟着廖雪峰学JavaScript（一）

发表于 2020-03-29 |

1、数据类型

JavaScript 的数据类型包含：Number , 字符串, 布尔值, 比较运算符, null 和 undefined , 数组, 对象, 变量
【注意】
①不区分整数和浮点数，统一用 Number 表示
② == 会自动转换数据类型，不推荐使用
=== 不会自动转换数据类型，如果数据类型不一致，直接返回 false, 如果一致才比较
③Number中特殊值
NaN 表示 Not a Number，当无法计算结果时用NaN表示
这个特殊的 Number 与其他值都不相等，包括它自己

唯一能判断 NaN的方法是isNaN()函数

Infinity 表示无限大，当数值超过Number所能表示的最大值时，就表示为 Infinity
④浮点数比较 (*其他编程语言一般也会有此问题)

原因：计算机无法精确表示无限循环小数，浮点数在运算过程中会产生误差
修改：计算它们之差的绝对值，看是否小于某个阈值

⑤null 和 undefined
null 表示 “空”，0 表示数值，''表示长度为0的字符串
undefined 表示未定义
[区分两者意义不大。大多数情况下，我们都应该使用 null。undefined 仅仅在判断函数参数是否传递的情况下有用]
⑥数组索引超出范围，返回 undefined
⑦动态语言

[区分静态语言，在定义变量时必须指定变量类型，如果赋值类型不匹配就会报错]
⑧strict 模式
在 strict 模式下运行的 JavaScript 代码，强制通过 var 申明变量，未使用 var 将导致运行错误
启用 strict 模式的方法是在JavaScript 代码的第一行写上：

浏览器默认不启用，如果浏览器不支持，会把 'use strict' 当成一个字符串

不用 var 申明的变量会变为全局变量，为了避免难以调试的问题，所有的代码都应该使用 strict 模式

2、字符串 ('' 或 "")

【注意】
①多行字符串
ES6标准新增，使用反引号 ` ` 替代每行使用\n来表示多行字符串

②模板字符串
ES6标准新增，使用反引号 ` ` 包裹变量，解决变量多时使用 + 拼接的麻烦

③获取某个指定位置的字符，使用类似 Array 的下标操作 （eg. str[0]）
超出范围的索引不会报错，返回 undefined
④字符串不可改变。如果对某个索引赋值，不会报错，但也没有任何效果
⑤操作字符串

3、数组

①数组长度 eg. arr.length
【特别注意】直接给 Array 的 length 赋新值会导致 Array 大小的变化

②通过索引赋值，索引超出返回，也会引起 Array 大小的变化

*大多数语言不允许直接改变数组大小。所以不建议直接修改 Array 的大小，访问索引时要确保索引不会越界
③操作数组

4、对象

①键值对用,隔开【特别注意】最后一个键值对不需要在末尾加,，如果加了有些浏览器可能将报错
②属性名如果包含特殊字符，需要用' '括起来，且需要用['xxx']来访问

*实际上对象的所有属性都是字符串
③访问不存在的属性，返回 undefined
④操作对象

5、条件判断

①if...else... 如果语句块只包含一条语句，可以省略{}

*不建议省略{}。省略{}的危险之处在于，如果后来添加一些语句，却忘了{}，将导致某些语句不在 if...else... 的控制范围内
*JavaScript 把 null, undefined, 0, NaN 和空字符串'' 视为 false，其他的均视为 true

6、循环

①for 循环的3个条件可以省略，但必须用 break 语句退出循环，否则就是死循环

②for...in
遍历对象，可以结合hasOwnProperty()来过滤对象继承的属性

遍历数组，得到是下标(字符串,不是Number)

7、Map 和 Set

①Map 键值对结构，键可以是 Number 或者其他数据类型
ES6标准新增。避免对象的键只能是字符串
初始化 Map 需要一个二维数组，或者直接初始化一个空 Map

②Set 一组 key 的集合，但不存储 value。没有重复的 key
ES6标准新增
初始化 Set 需要一个数组，或者直接初始化一个空 Set

8、iterable 类型

ES6标准新增，Array, Map, Set 都属于 iterable 类型
iterable 类型的集合可以通过 for...of 循环(ES6标准新增)来遍历，更好的方式是forEach方法(ES5.1标准引入)
for...of 循环

for...of 和 for...in 的区别
for...in 其实遍历的是对象的属性名称。数组实际上也是一个对象，它的每个元素的索引被视为一个属性
for...of 只循环集合本身的元素

forEach 方法 (仅3个参数有用; 超过3个的参数值为 undefined; 只使用1个参数的值为集合元素值)
